A Free Online Zipper Utility created as a Demo Project.

Created using PHP ZipArchive class.
As a Demo Project only upto 5 files are supported with a max size of 100KB each. This protects the server from running out of resources. Download links last for about 24 hours, as a cleanup mechanism has been setup to clean uploaded files after every 24 hours. You can choose to either enable compression(deflate) or disable compression(store).
